# F1 Statistics App

Приложение для просмотра статистики сезона Формулы 1.

## Функционал

В приложении реализовано 4 основных модуля:

1. Гонки
- Календарь этапов сезона.
- Детальные результаты гонок (позиции, очки, время).
- Информация о трассах.

2. Пилоты
- Список всех гонщиков.
- Профиль пилота со статистикой.

3. Таблица
- Турнирная таблица пилотов (Driver Standings).
- Кубок конструкторов (Constructor Standings).

4. Авторизация
- Вход и регистрация.
- Пользователи сохраняются в Room
